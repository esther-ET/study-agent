# Research Agent 设计文档

## 项目概述

- **项目名称**: Research Agent
- **项目类型**: AI Agent 应用
- **技术栈**: LangGraph + LangSmith
- **目标**: 基于 Semantic Scholar API 的智能学术论文搜索智能体
- **使用场景**: 本地开发调试 + 学习 Agent 开发用于简历

---

## 功能需求

### 1. 核心功能

| 功能 | 描述 |
|------|------|
| 智能搜索 | 输入中文或英文关键词，语义模糊匹配相关论文 |
| 关键词提取 | 自动从自然语言中解析出多个语义概念 |
| 年份限制 | 默认近5年，支持用户动态指定 |
| 排序规则 | 按时间和期刊分量排序 |
| 结果展示 | Markdown 格式：标题、年份、摘要（中英对照）、链接 |

### 2. 后续处理

| 功能 | 描述 |
|------|------|
| 导出 BibTeX | 导出选中论文为 BibTeX 格式 |
| 导出 CSV | 导出选中论文为 CSV 格式 |
| 生成摘要 | 对关键论文生成简短总结 |

### 3. 可视化与追溯

- LangSmith 集成：记录每个 Agent 节点的执行过程
- 输出 trace 链接，方便追溯完整推理链
- 备用方案：本地 JSON logging

## 整体架构

```
用户输入（CLI）
    ↓
┌─────────────────────────────────────────────┐
│           Agent Brain (StateGraph)           │
│  ┌─────────┐  ┌──────────┐  ┌─────────────┐  │
│  │ Intent  │→ │ Search   │→ │ Formatter   │  │
│  │ Parser  │  │ Executor  │  │ (Markdown)  │  │
│  └─────────┘  └──────────┘  └─────────────┘  │
│       ↓             ↓               ↓        │
│  ┌─────────────────────────────────────────┐  │
│  │         LangSmith Tracing               │  │
│  └─────────────────────────────────────────┘  │
└─────────────────────────────────────────────┘
    ↓
可选: Export (BibTeX/CSV) / Summarizer
    ↓
Markdown 格式输出
```

---

## 核心组件

| 组件 | 职责 | 输入 | 输出 |
|------|------|------|------|
| `IntentParser` | 解析用户输入，提取搜索意图 | 自然语言 | 结构化查询参数 |
| `SearchExecutor` | 调用 Semantic Scholar API | 查询参数 | 原始论文数据 |
| `ResultFormatter` | 格式化输出 Markdown | 原始数据 | Markdown 列表 |
| `ExportHandler` | 导出 BibTeX/CSV | 选中论文 | 文件 |
| `Summarizer` | 生成论文摘要 | 论文内容 | 摘要文本 |

## 状态流设计

```python
search_state = {
    "user_input": str,          # 用户原始输入
    "parsed_query": dict,       # 解析后的查询参数
    "year_filter": int,         # 年份限制（默认近5年）
    "sort_by": str,             # 排序方式（时间+期刊分量）
    "papers": list,             # 搜索结果
    "selected_papers": list,    # 用户选中的论文
    "export_format": str,       # 导出格式
    "tracing_url": str,         # LangSmith trace URL
}
```

## 技术选型

| 类别 | 选择 | 理由 |
|------|------|------|
| Agent 框架 | LangGraph | 状态机架构天然适配 Agent 循环，代码内聚、可视化、支持 LangSmith tracing |
| 数据源 | Semantic Scholar API | 学术论文数据丰富，支持无认证访问和 API Key 认证 |
| 追踪工具 | LangSmith | 可视化 Agent 推理过程，方便调试和简历展示 |
| 备用日志 | 本地 JSON logging | 不依赖外部服务时也可追溯 |
| 输出格式 | Markdown | 标题、年份、摘要（中英对照）、链接 |

## 设计决策记录

| 决策 | 选择 | 原因 |
|------|------|------|
| Agent 框架 | LangGraph | 相比 LangChain，架构更清晰，适合 Agent 循环，支持可视化 tracing |
| 数据源 | Semantic Scholar | 学术论文数据丰富，支持无认证访问 |
| 追踪方案 | LangSmith | 可视化推理过程，方便调试和简历展示 |
| 输出格式 | Markdown | 标题、年份、摘要（中英对照）、链接 |
| 关键词匹配 | 语义模糊匹配 | 不手动指定 AND/OR，自动解析意图 |
| 默认排序 | 时间 + 期刊分量 | 兼顾相关性和学术质量 |
| 年份默认 | 近5年 | 保持结果时效性 |
